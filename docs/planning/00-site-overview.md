## Design Elements

### Color Scheme
- **Primary Green:** #8ec63f (accent color for buttons, highlights)
- **Dark Footer:** #313233
- **Blue Secondary:** #0760ad, #326b93

### Layout Structure
1. **Header** - Logo and sticky navigation bar
2. **Hero Section** - Service category cards with images
3. **Featured Services** - Image/text blocks for each service area
4. **About Section** - Company overview and mission
5. **Footer** - Contact info, quick links, copyright

### Typography
- Clean, professional sans-serif fonts
- Headers in uppercase for emphasis
- Body text in standard case

### Visual Elements
- Service category cards with hover effects
- Full-width image sections
- Grid layout for project listings
- Icon-based service indicators
